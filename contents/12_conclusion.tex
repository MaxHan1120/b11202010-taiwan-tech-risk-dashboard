\section{Conclusion}
\label{sec:conclusion}

This project demonstrates how public financial data can be refined into a monetizable data product. The target customer is a Taiwan retail investor who follows semiconductor and technology stocks and needs a faster way to monitor risk. The product creates value by compressing scattered market and fundamental signals into an explainable dashboard.

The technical system implements the full data product loop: ingestion, storage, processing, API delivery, and frontend consumption. The risk score is intentionally rule-based and transparent, combining volatility, drawdown, abnormal volume, valuation, and fundamental deterioration. The dashboard then exposes those scores through overview, stock detail, factor ranking, and portfolio watchlist views.

The strongest business case is not that the product can beat the market. The stronger claim is that it can save research time, reduce missed risk signals, and help users prioritize deeper analysis. The main remaining risks are trust, data licensing, competition, and the need to validate willingness to pay with real users.

The next step would be to run a small user validation study with Taiwan retail investors, deploy the prototype publicly, and measure which features create repeat usage: daily ranking, alerts, portfolio risk, or stock-detail explanations.
